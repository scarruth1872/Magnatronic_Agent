  - Creative problem-solving
- **Tools:**

  - Creative algorithms

### Knowledge Agent
- **Primary Functions:**
  - Information management
  - Query processing
  - Knowledge base maintenance
- **Tools:**

### Monitoring Agent
- **Primary Functions:**

- **Tools:**
